Bayesian Blocks
=======================================

This program computes Bayesian Blocks, including a visualisation, 
from a sequence of event times.

Usage
------

$ ./bb.py mydatafile

* Will create a plot "mydatafilebblocks.pdf"
* And a data file "mydatafilebblocks.txt", containing three columns:
  * start time
  * stop time
  * rate
